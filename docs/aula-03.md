# Instalar o Body Parser
```bash 
npm i --save body-parser
```

# Instruções Gerais

- Mapear todas das rotas das Tarefas: `routes/tasks.js`
- Implementar os métodos GET, POST, DELETE e PUT
- Refactoring no `libs/middlewares.js` para adicionar o `body-parser`
- Replicar API Tasks para Usuários: `routes/users.js`
- Testes as rotas da API utilizando POSTMAN
